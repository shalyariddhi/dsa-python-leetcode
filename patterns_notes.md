# DSA Pattern Notes

## Hashing
- Store seen elements for O(1) lookup.
- Used in: Two Sum, Contains Duplicate, Valid Anagram.

## Stack
- Last In First Out (LIFO).
- Used when matching pairs.
- Example: Valid Parentheses.

## Sliding Window
- Expand the window with the right pointer.
- Shrink when the condition becomes invalid.
- Example: Longest Substring Without Repeating Characters.

## Prefix Sum
- Accumulate values while scanning.
- Example: Running Sum.

## State Tracking
- Track best value while scanning.
- Example: Best Time to Buy and Sell Stock.
